# Meteo-M

A small, client-side weather app built with React and TypeScript. Meteo-M fetches location and weather data from the Open-Meteo API and displays the current weather plus a 5-day forecast.

Features
- Search for cities using geocoding
- Use browser geolocation to show local weather (when available)
- Current conditions (temperature, wind, precipitation)
- 5-day daily forecast

Data
- Weather and forecast data are provided by the Open-Meteo API (https://open-meteo.com).
